파일 저장 방식

1. 텍스트 파일 : 아스키 코드 값으로 저장되는 파일 2. 이진 파일 : 사용자가 정의한 데이터 형으로 저장된 파일 공통점: 0과 1로 저장된 이진 데이터 차이점: 저장하는 방식과 읽는 방식

**파일 입출력 함수(파일 형식과 상관없이 공용으로 사용하는 함수) FILE * fopen(파일이름,모드) fclose(FILE * stream) feof(FILE * stream) fscanf(파일 포인터,"서식 문자", &읽어온 것 담을 변수) 파일 읽을 때 fprintf(파일 포인터, "~~~") 파일에 저장할 때

**모드
1.파일 접근 모드
t: 텍스트 파일 모드로 파일 열기(생략가능) b: 이진 파일 모드로 파일 열기
2.입출력 모드

r:파일 읽기
w:파일 쓰기 파일이 없으면 새로 생성하거나 있는 경우는 덮어 쓰기
a:기존 파일에 내용을 추가, 만약 파일이 없으면 새로 생성
r+: 파일이 존재할 때만 읽기와 쓰기 기존 파일이 있는 경우 쓰기에서 덮어쓰기 함 파일이 없는 경우 새로운 파일 생성 못함
w+: r+와 동일하며 파일이 없으면 새로운 파일 생성
a+: 파일 읽기,추가 파일이 없는 경우 새로운 파일 생성
파일 mode 결합 순서: 입출력 모드+ 파일 접근 모드+ '+'
